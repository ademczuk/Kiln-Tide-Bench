Verdict

Cost, quality, and when the boardroom is worth it

This prompt did not need a committee to get to “good.” It only started paying for extra brains once the bar moved from shipping a competent static page to auditing trade-offs and upgrading imagery.

Solo won the default case

For a one-page brochure site, the best returns came from direct single-agent execution. Shimmer and Claude CLI sat in the 1.5 to 3 minute band with dramatically lower orchestration cost.

Boardroom is a premium path

The boardroom run makes sense when the operator values debate, auditability, and layered improvements more than raw speed. It is not a cheap way to get a first draft.

Swarm failure is signal, not noise

KCS burning 513 seconds for an empty artifact is the cleanest warning on this page: decomposition overhead can be actively harmful when the task is one self-contained file.
